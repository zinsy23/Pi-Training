## Pi-Training
Install this program and be able to memorize Pi!
To download an individual file, like [Pi Training Setup.exe](https://github.com/zinsy23/Pi-Training/raw/master/Pi%20Training%20Setup.exe), click on the file and find the download button above.
The source code folder contains the source code for the program and timers.
There is also a separate file with comprehensive commentary on the source code. They should run the same as the regular source code.
The default preferences.ini file is also located here!

You may have to disable your anti-virus and "run anyways" on Windows SmartScreen (Windows 8 and newer.)
